## Deskripsi
Aplikasi web sederhana menggunakan PHP Native dan MySQL untuk mengelola Kategori Buku dengan fitur CRUD (Create, Read, Update, Delete) lengkap dan validasi server-side.

## Cara Instalasi
1. Pindahkan folder proyek ke dalam folder `htdocs` (jika menggunakan XAMPP).
2. Buat database baru di phpMyAdmin.
3. Import file `uts_perpustakaan_60324073.sql` ke database tersebut.
4. Sesuaikan konfigurasi database pada file `config/database.php`.
5. Buka aplikasi di browser: `http://localhost/uts_60324073/`
